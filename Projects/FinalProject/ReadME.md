# 🎬 سیستم مدیریت فروش بلیت سینما

## 📖 معرفی پروژه

این پروژه یک برنامه کنسولی ساده با زبان Java است که فرآیند فروش بلیت سینما را شبیه‌سازی می‌کند. برنامه اطلاعات مشتریان را دریافت کرده، قیمت بلیت را بر اساس نوع فیلم و سن مشتری محاسبه می‌کند و در پایان گزارش کاملی از فروش ارائه می‌دهد.

---

## ✨ امکانات

✅ ثبت اطلاعات مشتریان

✅ اعتبارسنجی ورودی‌ها

✅ محاسبه خودکار قیمت بلیت

✅ اعمال ۵۰٪ تخفیف برای:
- 👶 کودکان زیر ۱۲ سال
- 👴 افراد بالای ۶۰ سال

✅ محاسبه مجموع فروش

✅ شمارش تعداد بلیت‌های فروخته شده در هر دسته

✅ نمایش مسن‌ترین مشتری ثبت‌شده

---

## 🎥 دسته‌بندی فیلم‌ها

| نوع فیلم | قیمت بلیت |
|-----------|------------|
| 👶 کودکان (Children) | 20,000 تومان |
| 🔫 اکشن (Action) | 40,000 تومان |
| 🎭 درام (Drama) | 30,000 تومان |
| 📚 مستند (Documentary) | 25,000 تومان |

---

## 🛡 قوانین اعتبارسنجی

- نام مشتری باید حداقل ۳ کاراکتر داشته باشد.
- سن باید بین ۷ تا ۷۰ سال باشد.
- نوع فیلم فقط می‌تواند یکی از موارد زیر باشد:
  - children
  - action
  - drama
  - documentary
