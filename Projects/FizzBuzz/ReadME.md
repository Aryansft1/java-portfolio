# 🎮 بازی FizzBuzz

## 📖 معرفی پروژه

این پروژه پیاده‌سازی بازی معروف FizzBuzz با زبان Java است. برنامه یک عدد از کاربر دریافت می‌کند و از ۱ تا آن عدد را بررسی می‌کند. سپس بر اساس قوانین بازی، خروجی مناسب را نمایش می‌دهد.

---

## 🎯 قوانین بازی

برای هر عدد:

🔹 اگر مضرب ۳ باشد:
```text
Fizz
```

🔹 اگر مضرب ۵ باشد:
```text
Buzz
```

🔹 اگر مضرب هر دو عدد ۳ و ۵ باشد:
```text
FizzBuzz
```

🔹 در غیر این صورت خود عدد نمایش داده می‌شود.

---

## ✨ ویژگی‌ها

✅ دریافت عدد از کاربر

✅ استفاده از حلقه برای پیمایش اعداد

✅ تشخیص مضرب‌های ۳ و ۵

✅ نمایش خروجی مطابق قوانین FizzBuzz

✅ پیاده‌سازی ساده و خوانا
