# Prime in Java 🧮

## توضیح
بک کد ساده برای پیدا کردن عدد اول 

## ویژگی‌ها
- ابتدا برنامه باید میانگین عدد واردشده با عدد 13 را محاسبه کند.

- سپس بررسی کند که آیا عدد واردشده اول است یا خیر.

- اگر عدد اول باشد، فقط نتیجه میانگین و اول بودن عدد نمایش داده می‌شود.
- اگر عدد اول نباشد، علاوه بر موارد بالا، باید بزرگ‌ترین مقسوم‌علیه عدد (به‌جز خودش) نیز نمایش داده شود
- استفاده از متد ها 

## اجرا
کلاس Main را اجرا کنید.
