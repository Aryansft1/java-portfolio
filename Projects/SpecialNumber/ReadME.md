# 🔢 تولید الگوهای عددی

## 📖 معرفی پروژه

این پروژه یک برنامه کنسولی با زبان Java است که بر اساس عدد وارد شده توسط کاربر، مجموعه‌ای از الگوهای عددی تولید و نمایش می‌دهد.

برنامه برای هر عدد از ۱ تا n رفتار متفاوتی دارد:

- اگر عدد زوج باشد، مربع آن نمایش داده می‌شود.
- اگر عدد فرد باشد، اعداد فرد از ۱ تا همان عدد چاپ می‌شوند.

---

## ✨ امکانات

✅ دریافت عدد از کاربر

✅ تشخیص اعداد زوج و فرد

✅ محاسبه مربع اعداد زوج

✅ تولید دنباله اعداد فرد

✅ استفاده همزمان از حلقه For و While

## 📋 نمونه اجرا

ورودی:

```text
5
```

خروجی:

```text
1: 1

3: 1 3
4: 16
5: 1 3 5
```

---

## 🎯 منطق برنامه

🔹 برای اعداد زوج:

```text
عدد × عدد
```

مثال:

```text
4 → 16
6 → 36
```

🔹 برای اعداد فرد:

تمام اعداد فرد از ۱ تا همان عدد چاپ می‌شوند.

مثال:

```text
5 → 1 3 5
7 → 1 3 5 7
```
